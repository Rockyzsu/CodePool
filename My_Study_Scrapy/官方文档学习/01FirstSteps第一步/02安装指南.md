# 安装指南

## 安装环境
scrapy的安装依赖于python,需要python > 2.7 或者python > 3.3

## 安装方法
使用conda你可以使用如下命令安装
```
conda install -c conda-forge scrapy
```

使用pip你可以使用如下命令安装
```
pip install  scrapy
```

## scrapy的常用依赖包
* lxml:一个官方的xml和html 解析器
* parsel:一个html/xml数据提取库基于lxml
* w3lib:一个处理URL和网页编码的多功能的帮助库
* twisted:一个异步网络框架
* cryptograpyh和pyOpenSSL: 网络级的安全需要
